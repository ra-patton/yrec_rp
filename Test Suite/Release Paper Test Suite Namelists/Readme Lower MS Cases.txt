We provide five sample cases for lower MS models that use different choices for surface boundary conditions equations of state (m0p1 feh+0.0 allard ZAMS; m0p3 feh+0.0 allard ZAMS; m0p1 feh+0.0 base ZAMS; m0p3 feh+0.0 scveos ZAMS; m0p3 feh+0.0 yaleeos ZAMS) in the test suite.

m0p1 feh+0.0 allard ZAMS uses the base physics suite, except for the usage of the allard model atmospheres for the SBC.
m0p3 feh+0.0 allard ZAMS uses the base physics suite, except for the usage of the allard model atmospheres for the SBC.
m0p1 feh+0.0 base ZAMS uses the base physics suite in a domain that is challenging for stellar models.
m0p3 feh+0.0 scveos ZAMS uses the base physics suite, except for the usage of the SCV EoS instead of the OPAL EoS.
m0p3 feh+0.0 yaleeos ZAMS uses the base physics suite, except for the usage of the Yale EoS instead of the OPAL EoS
