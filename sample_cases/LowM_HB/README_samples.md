# Core He-burning models

| Model Name     | Description    |
| ------------ | -------------- |
| `Solar_m1p0feh+0p0_GN93_MESA_TAHB`      | A solar-mass core He-burning model evolved from a zero-age horizontal branch seed to central Y = 0.0001. Mixing length of 2.13 is assumed. No semi-convection or overshooting. No diffusion, no rotation, and with a gray atmosphere. Note that abundances and associated atomic opacity table mixtures are GN93 not GS98 standard for other test cases, in order to be consistent with the seed model. The core mass is chosen to be consistent with a MESA model run with GN93 opacties. The core mass is chosen to be consistent with a MESA model with matched physics. The first run is a rescaling to the desired metallicity, the second run to the desired core mass, and the third run is the evolution run.|
| `Solar_m1p0feh+0p0_PARSEC_TAHB`      | A solar-mass core He-burning model evolved from a zero-age horizontal branch seed to central Y = 0.001. Mixing length of 1.71 is assumed. No semi-convection. Overshooting. No diffusion, no rotation, and with a gray atmosphere. Atomic opacities calculated with a GS98 mixture and abundances are GN93. The core mass is chosen to be consistent with a the non-rotating solar PARSEC v2.0 model. The first run is a rescaling to the desired metallicity, the second run to the desired core mass, and the third run is the evolution run.|
| `ZAHB_m?p?feh+0p0_TAHB`      | Base core He-burning models evolved from a zero-age horizontal branch seed to central Y = 0.0001. Mixing length of 1.71 is assumed. No semi-convection. Overshooting. No diffusion, no rotation, and with a gray atmosphere. Atomic opacities calculated with a GS98 mixture and abundances are GN93. Envelope overshooting is included to be consistent with PARSEC v2.0 mass-dependent values. The core mass is chosen to be consistent with the corresponding base case model grid core mass at the tip of the red giant branch. There are varying numbers of runs, which include first a metallicity rescaling, followed by a number of core mass rescaling run, and a final evolution run.|
| `ZAHB_m?p?feh+0p0_diffcal_TAHB`      | Core He-burning models evolved from a zero-age horizontal branch seed to central Y = 0.0001 with diffusion-calibrated Y= 0.273 and Z = 0.0188. Otherwise identical to `ZAHB_m?p?feh+0p0_TAHB`.|
