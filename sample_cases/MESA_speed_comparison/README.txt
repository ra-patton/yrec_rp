#####################################################################
MESA vs. YREC speed tests
####################################################################

YREC models use the base model grid, solar mass, solar metallicity case. 

MESA models are run using MESA r24.08.1 and the inlist_speed provided 
in this directory. The run options for the pre-MS, MS< and post-MS MESA runs
are all together in the file; the user can comment in the option they with to run.
